▸ Alert lifecycle

States: NEW → ACK → WATCHING → CLOSED → (REACTIVATED). NEW alerts surface in Highlights if score-ranked top 10. ACK'd alerts move to the trigger table with age counter. WATCHING = ACK'd but threshold still breached. CLOSED when underlying condition resolves OR manual close with reason code. REACTIVATED if CLOSED alert re-fires within 4 weeks (anti-flapping). Each alert carries brand, state, dimension, severity, threshold, observed value, cohort, age, last-actioned-by, last-actioned-when. Format mirrors Salesforce case management conventions.
